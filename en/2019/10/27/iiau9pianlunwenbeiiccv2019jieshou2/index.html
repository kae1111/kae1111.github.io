9 Papers Were Accepted to ICCV 2019

Dr.Li Pei-Xia’s paper：GradNet: Gradient-Guided Network for Visual Object Tracking.

Yan Bin’s paper：Cooling-Shrinking Attack: Blinding the Tracker with Imperceptible Noises.

Dr. Liu Zi-Mo’s paper: Deep Reinforcement Active Learning for Human-In-The-Loop Person Re-Identification.

Dr. Zhang Ping-Ping’s paper：Cascaded Context Pyramid for Full-Resolution 3D Semantic Scene Completion.

Prof. Piao Yong-Ri’s paper：Depth-induced Multi-scale Recurrent Attention Network for Saliency Detection.

Yan Bin’s paper: Skimming-Perusal’ Tracking: A Framework for Real-Time and Robust Long-term Tracking.

Dr. Zhang Lu’s paper: Fast Video Object Segmentation via Dynamic Targeting Network.

Zeng Yi’s paper: Towards High-Resolution Salient Object Detection.

Zeng Yu’s paper: Joint learning of saliency detection and weakly supervised semantic segmentation.

Wang Tian-Tian’s paper: Deep Learning for Light Field Saliency Detection.

Congratulations to the above students and teachers！
